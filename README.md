# BugLab
### A Simple Bug Management App
Bora Kucukkara & Yunus Emre Altanay

#### Frameworks: Vue.js / Vuex / Laravel

## Features
- Report Bugs
- Set Priority
- Custom Categories
- Archive
- Filter by Date, Status, Priority or Category
- Simply double click on report to edit
---
